# 等时线地图 | Isochrone Map

基于真实路网数据的等时线分析工具，支持全球 15+ 主要城市，采用三级降级引擎确保高可用。

## ✨ 功能特点

- 🗺️ **真实路网** - Valhalla (OSM) → ORS → 几何近似 三级降级引擎，确保始终可用
- 🚗 **四种交通方式** - 驾车、步行、骑行、公交/地铁
- ⏱️ **灵活时间范围** - 5–120 分钟，5 分钟间隔多带等时线
- ✈️ **机场标记** - 显示各城市主要机场位置及交通时间
- 🚂 **火车站标记** - 显示高铁站、主要火车站位置
- 🌍 **全球城市** - 覆盖北京、上海、广州、深圳、中国香港、东京、首尔、新加坡、伦敦、巴黎、纽约、柏林、悉尼、迪拜、曼谷 15 城
- 🎨 **暗色主题** - CARTO Dark Matter 底图 + 渐变等时线 + 现代化 UI

## 🚀 快速开始

### 方法一：双击启动
双击 `启动等时线地图.bat` 即可启动

### 方法二：命令行启动
```bash
cd isochrone-map
python server.py
```

然后打开浏览器访问: http://localhost:8901

## 🎯 使用方法

1. **选择城市** - 在搜索框中输入城市名称，或从下拉列表中选择
2. **选择交通方式** - 点击驾车🚗、步行🚶、骑行🚴或公交🚇按钮
3. **设置时间范围** - 拖动滑块设置 5–120 分钟的时间范围
4. **生成等时线** - 点击"生成等时线"按钮
5. **查看结果** - 地图上会显示不同颜色的等时线区域，反映真实路网可达范围

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript, Leaflet.js 1.9.4
- **后端**: Python 3, http.server
- **等时线引擎**: Valhalla (OSM 公共实例) → OpenRouteService → 服务端几何近似
- **地图瓦片**: CARTO Dark Matter

## 🔧 配置说明

### 等时线引擎（三级降级）

| 优先级 | 引擎 | 说明 |
|--------|------|------|
| 1 | **Valhalla** | OpenStreetMap 公共实例，无需 API Key，路网质量优 |
| 2 | **ORS** | OpenRouteService，需设置环境变量 `ORS_API_KEY` |
| 3 | **几何近似** | 服务端路网感知多边形（72 顶点 + 正弦波模拟），离线可用 |

### OpenRouteService API Key（可选）

如需启用 ORS 作为备用引擎，设置环境变量：

```bash
# Windows
setx ORS_API_KEY "your_api_key_here"

# Linux/macOS
export ORS_API_KEY="your_api_key_here"
```

前往 https://openrouteservice.org/dev/#login 免费注册获取 API Key。

## 🌟 特色功能

### 三级降级引擎
- **Valhalla**：使用 OSM 公共实例，无需配置，提供真实路网等时线
- **ORS**：作为备用引擎，需 API Key，免费额度每天 2000 次
- **几何近似**：当外部 API 不可用时自动启用，72 顶点路网感知多边形

### 机场信息
- 显示机场位置标记（橙色菱形）
- 点击标记查看机场代码、距离、交通时间

### 火车站信息
- 显示火车站位置标记（绿色圆形）
- 点击标记查看车站名称、类型（高铁/普速）

### 等时线可视化
- 5 分钟间隔的渐变色等时线
- 从中心向外扩散的多层效果
- 动态图例显示当前时间段

## 📝 注意事项

1. Valhalla 公共实例可能因网络原因不可用，系统会自动降级
2. 几何近似模式为路网感知多边形，精度低于真实路网但优于简单圆形
3. 建议使用 Chrome 或 Edge 浏览器获得最佳体验
